Excel Dashboard Challenges & Approach

Challenges

The main problem with the Excel dashboard project was dealing with raw data that wasn't ready for analysis. Some issues included columns that were named confusingly and data that wasn't in the right format, like numbers stored as text.

Approach

To fix these issues, I started by renaming the columns to make them clearer and easier to work with. Then, I changed the data types, turning text that should be numbers into actual numbers. Finally, I set up several pivot tables to help analyze the data better and draw useful insights for the dashboard.
