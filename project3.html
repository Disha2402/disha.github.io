Challenges Faced & Problem-Solving Approach

Challenges

A significant challenge in this project was using social media data effectively to reveal meaningful insights. Specifically, visualizing social media usage by age group was complex. I struggled with understanding how to create multiple graphs at once, which led me to use a Python script to generate these visualizations.

Approach

As with previous projects, I started by creating a rough layout in Figma. This practice proved invaluable not only for planning the dashboard's structure but also for selecting a color palette. To enhance the effectiveness of the dashboard, I carefully selected the right types of visualizations that best matched the data characteristics. This involved analyzing the data to determine which visual formats would most clearly and effectively convey the key insights, such as using bar charts for demographic comparisons and line graphs for trend analysis.
